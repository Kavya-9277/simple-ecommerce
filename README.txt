# Simple E-commerce Website

A simple e-commerce website created using HTML, CSS, and JavaScript that allows users to browse products, add them to the shopping cart, and place an order. 

## Features

- *Product Display*: A list of available products with their prices.
- *Add to Cart*: Users can add products to their shopping cart.
- *Cart Management*: Users can see a list of products added to the cart and the total price.
- *Place Order*: Users can place an order if they have products in their cart.

## Technologies Used

- *HTML*: Structure of the web page.
- *CSS*: Styling the website with a clean, modern design.
- *JavaScript*: Functionality for adding products to the cart and placing the order.

## How to Use

1. Clone the repository to your local machine.
    bash
    git clone https://kavya-9277.github.io/simple-ecommerce/
    
2. Open the index.html file in any modern browser.

3. Browse products and add them to the cart.

4. View the cart and place your order.
